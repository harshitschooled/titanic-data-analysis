# Titanic Data Analysis

## Project Overview
This project performs data preprocessing and exploratory data analysis on the Titanic dataset.

## Tasks Performed
- Missing value handling
  - Mean Imputation
  - Median Imputation
  - Mode Imputation
- Data Visualization using Seaborn and Matplotlib
- Label Encoding
- Feature Analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Google Colab

## Dataset
Titanic Dataset
